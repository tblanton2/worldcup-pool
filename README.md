# 2026 World Cup Pool

A real-time 8-person World Cup pool app built with React, Vite, and Firebase Firestore.

## Deploy

1. Push this repo to GitHub
2. Import the repo on vercel.com
3. Vercel auto-detects Vite — no configuration needed
4. Click Deploy — done.

## Local dev

```bash
npm install
npm run dev
```
